🧹 Text Preprocessing

Text preprocessing transforms raw Hindi text into a clean, structured format that can be effectively processed by Natural Language Processing (NLP) models.

Concepts Covered

  1. Sentence Segmentation
  2. Tokenization
  3. Case Normalization
  4. Stop-word Removal
  5. Stemming
  6. Lemmatization
  7. Noise Removal
  8. Text Cleaning

1. Sentence Segmentation

Definition

Sentence Segmentation is the process of dividing a large text document into individual sentences. It identifies sentence boundaries using punctuation marks such as full stops, question marks, and exclamation marks.

Purpose

The main purpose is to organize large text into manageable units before further NLP processing.

Working Principle

Input

भारत ने विश्व कप जीता। टीम ने शानदार प्रदर्शन किया। पूरे देश में खुशी का माहौल है।

Output

भारत ने विश्व कप जीता।

टीम ने शानदार प्रदर्शन किया।

पूरे देश में खुशी का माहौल है।

Example

A Hindi news article containing multiple statements is divided into separate sentences for easier analysis.

Advantages

  • Improves text organization
  • Simplifies preprocessing
  • Helps sentence-level analysis

Limitations

  • Abbreviations may cause incorrect segmentation.
  • Complex punctuation may create errors.

Applications

  • News Classification
  • Machine Translation
  • Text Summarization
  • Chatbots

2. Tokenization

Definition

Tokenization is the process of splitting text into smaller units called tokens.

Purpose

Convert human-readable text into machine-processable units.

Working Principle

Input

भारत ने विश्व कप में शानदार जीत हासिल की।

Output

भारत | ने | विश्व | कप | में | शानदार | जीत | हासिल | की

Example

Words such as "क्रिकेट", "खिलाड़ी", and "मैच" help identify sports-related documents.

Advantages

  • Foundation of NLP
  • Helps feature extraction
  • Simplifies analysis

Limitations

  • Compound Hindi words may be difficult to tokenize.
  • Context is not preserved.

Applications

  • Search Engines
  • Chatbots
  • Text Classification

3. Case Normalization

Definition

Case normalization converts text into a consistent format by changing letters into lowercase or uppercase.

Purpose

Avoid treating words with different letter cases as different words.

Example

↓

Hindi itself has no uppercase/lowercase distinction, but this step is useful in mixed Hindi-English datasets.

Advantages

  • Reduces vocabulary size
  • Improves consistency

Limitations

  • May remove useful distinctions in some languages.

Applications

  • Text Classification
  • Search Systems
  • Information Retrieval

4. Stop-word Removal

Definition

Removes frequently occurring words that provide little information for classification.

Example

भारत ने क्रिकेट मैच में शानदार जीत हासिल की है।

↓

भारत क्रिकेट मैच शानदार जीत हासिल

Advantages

  • Reduces dimensionality
  • Faster processing
  • Improves efficiency

Limitations

  • Some stop words may carry useful meaning.

Applications

  • Search Engines
  • Sentiment Analysis
  • News Classification

5. Stemming

Definition

Stemming reduces words to their root form by removing prefixes and suffixes.

Example

खेलना → खेल
खेलते → खेल
खेला → खेल

Advantages

  • Fast processing
  • Reduces feature space

Limitations

  • Root words may not always be linguistically correct.

Applications

  • Information Retrieval
  • Text Classification

6. Lemmatization

Definition

Lemmatization converts words into their meaningful dictionary forms.

Example

बच्चों → बच्चा
लड़कों → लड़का

Advantages

  • More accurate than stemming
  • Preserves semantics

Limitations

  • Computationally expensive.

Applications

  • Machine Translation
  • Chatbots
  • NLP Pipelines

7. Noise Removal

Definition

Noise removal eliminates emojis, punctuation, URLs, symbols, hashtags, and extra spaces.

Example

भारत ने मैच जीता!!! 😊

↓

भारत ने मैच जीता

Advantages

  • Cleaner dataset
  • Better model performance

Limitations

  • Some symbols may carry useful information.

Applications

  • Social Media Analysis
  • News Processing

8. Text Cleaning

Definition

Text cleaning combines multiple preprocessing techniques to prepare text for NLP tasks.

Example

Raw Text

भारत!!! ने ## मैच जीता 😊

Clean Text

भारत ने मैच जीता

Advantages

  • Improves accuracy
  • Standardizes data

Limitations

  • Excessive cleaning may remove useful information.

Applications

  • Text Classification
  • Information Retrieval
