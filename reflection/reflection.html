📝 Reflection Notes

👨‍💻 Mayank's Reflection

Working on this project helped me understand the complete Natural Language Processing pipeline, beginning with text preprocessing and feature engineering. I explored techniques such as sentence segmentation, tokenization, stemming, lemmatization, Bag of Words, N-Grams, and TF. Developing workflow diagrams and studying applications such as search engines and machine translation improved my understanding of how NLP is applied in real-world systems. This project also enhanced my skills in technical documentation, HTML/CSS development, and GitHub repository management.

👩‍💻 Mohit's Reflection

This project provided valuable experience in advanced language representation techniques including TF-IDF, Word Embeddings, Word2Vec, FastText, contextual embeddings, and transformer-based models. I learned how modern NLP systems perform Hindi text classification and explored applications such as text summarization, spam detection, and news classification. Researching MuRIL and industrial recommendation systems gave me practical insights into current AI technologies. Overall, this project strengthened my knowledge of Natural Language Processing, web-based documentation, and collaborative software development.
