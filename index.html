Hindi Text Classification using Natural Language Processing

A comprehensive knowledge repository demonstrating Natural Language Processing techniques for Hindi Text Classification.

📖 Get Started 📚 Explore Concepts

👋 Welcome

This repository is developed as part of the Natural Language Processing (NLP) course.

The project demonstrates how NLP techniques can be applied to classify Hindi text into predefined categories using modern Machine Learning and Deep Learning approaches.

🎯 Objectives

🧹 Text Preprocessing

Learn cleaning, tokenization, stop-word removal, stemming, and lemmatization.

📊 Feature Engineering

Understand Bag of Words, N-Grams, TF, IDF, and TF-IDF.

🤖 Language Representation

Study Word2Vec, FastText, Contextual Embeddings, and Transformer Models.

⚙️ Classification Workflow
